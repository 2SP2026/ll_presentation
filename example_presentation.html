Interactive HTML Presentations

Beautiful, Modern, and Interactive

Created with Python • Powered by HTML5 & CSS3

Features

Key Features

  • Slide navigation panel with quick jump access
  • Smooth animations and transitions
  • Keyboard navigation (Arrow keys, Space, Home, End)
  • Touch/swipe support for mobile devices
  • Progress bar and slide counter
  • Fullscreen mode (press 'F')
  • Multiple themes (dark, light, gradient)
  • Responsive design for all screen sizes

Easy to Use

Create presentations with simple Python code:

pres = PresentationGenerator(
    title="My Presentation",
    theme="dark"
)

pres.add_slide(
    title="Welcome",
    content="Hello World!"
)

pres.generate("output.html")
